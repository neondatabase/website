# Neon Auth roadmap

> The "Neon Auth roadmap" document outlines planned authentication features and enhancements for Neon, detailing the stages of development and anticipated release timelines for each feature.

## Source

   **Note** Standalone architectures: Architectures where frontend and backend are separate deployments (e.g., Create-React-App with a separate Node/Express backend) are not yet supported. Neon Auth uses HTTP-only cookies for secure session management, and these cookies cannot be securely shared between frontend and backend applications on different domains. We're actively working on this.
