# Platform integration guide

> The Platform Integration Guide outlines the steps for integrating Neon with various platforms, detailing configuration processes and compatibility requirements specific to Neon's database services.

## Source

- [Platform integration guide HTML](https://neon.com/docs/guides/platform-integration-intro): The original HTML version of this documentation

Learn how you can offer instant, managed Postgres databases to your users with Neon. This guide covers how to integrate your platform or service with Neon, set usage limits for your users, and more.
