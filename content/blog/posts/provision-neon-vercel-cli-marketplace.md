![Provision Neon from Vercel CLI](https://cdn.neonapi.io/public/images/pages/blog/provision-neon-vercel-cli-marketplace/cover.png)

The Vercel CLI now lets agents provision a database for your project with a single command from your CLI: `vercel integration add neon`

An agent can now:

- Initialize a Vercel project
- Add the Neon integration
- Retrieve the database connection string automatically
- Start writing backend logic that connects to your database

## Full Self Driving for Agents

Andrej Karpathy captured the current state of agent-provisioned infra well:

<BlogQuote quote="The reality of building web apps in 2025 is that it's a bit like assembling IKEA furniture. There's no “full-stack” product with batteries included, you have to piece together and configure many individual services." author="Andrej Karpathy" role="@karpathy" />

With this integration, the Vercel CLI keeps the entire flow in the terminal so agents can start building without browser logins, dashboard clicks, or manual handoff from you. This reduces steps for both humans and agents allowing both to instantly provision a Neon database for your next Vercel project.

[Learn more about what agents are capable of doing with Vercel CLI for Marketplace here.](https://vercel.com/marketplace/neon)
