## Postgres 18 for newly created Neon projects

![Create PG18 project](/docs/changelog/create_project_18.png)

Postgres 18 is now the default for newly created Neon projects. Neon continues to support Postgres 14, 15, 16, and 17, if you prefer to stick with those. For Neon's Postgres version support policy, see [Postgres Version Support](/docs/postgresql/postgres-version-policy).
