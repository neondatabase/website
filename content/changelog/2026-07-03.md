<details>
<summary>**Neon CLI**</summary>

- You can now install the Neon CLI from npm as `neon` instead of `neonctl`: `npm i -g neon` (or run it with `npx neon@latest`). All commands are now documented as `neon` rather than `neonctl`. If you already use `neonctl`, nothing changes: it's the same CLI, `neonctl` still works as a command, and no migration or re-authentication is needed. See the [Neon CLI install guide](/docs/cli/install).

</details>

<details>
<summary>**Compute updates**</summary>

- The advance notice period for planned updates on paid plans is now 3 days, down from 7 days. You'll see a notice in your project's settings (**Settings** > **Updates**) at least 3 days ahead of a planned update, and Scale plan customers receive an email notification 3 days in advance. To learn more, see [Updates](/docs/manage/updates).

</details>
