## Two-factor authentication

Neon now supports **two-factor authentication (2FA)** for your account. Once enabled, you enter a 6-digit code from your authenticator app (for example, Google Authenticator, Authy, or 1Password) each time you log in. Set it up from your profile menu: **Account settings** → **Set up two-factor authentication**, then scan the QR code and verify with a code from your app. Removal is delayed by 24 hours for security. For details, see [Two-factor authentication](/docs/manage/accounts#two-factor-authentication).

## Data API Advisors

**Data API Advisors** is a new tab on the **Monitoring** page that helps you secure and tune your database when you use the [Neon Data API](/docs/data-api/overview), Neon's HTTP interface for querying your database from browsers, serverless functions, and edge runtimes. Because the Data API exposes your schema directly over HTTP, the RLS policies and security you define on your schema are critical. The advisors scan your database and report security and performance issues (such as missing RLS, sensitive columns exposed, or unindexed foreign keys) with severity levels and recommended fixes.

In the Neon Console, go to **Monitoring > Data API Advisors** for your branch to run a scan and view issues grouped by category.

![Data API advisor](/docs/data-api/data-api-database-advisor-monitor.png)

For details, see [Data API Advisors](/docs/data-api/database-advisor).

## New guide: AI-driven incident response with Vercel and Neon MCP

A new guide shows how to combine the [Vercel MCP server](https://vercel.com/docs/agent-resources/vercel-mcp) with the [Neon MCP server](/docs/ai/neon-mcp-server) so AI agents can diagnose production errors, validate database fixes on a Neon branch, and open pull requests with code changes. Use it with Claude Code or any agent that supports MCP. See the guide: [AI-driven incident response with Vercel and Neon MCP](/guides/vercel-neon-mcp).

## Multiple free organizations and members on the Free plan

Free plan limits have been updated: you can create **multiple free organizations** (up to 1,000 per account) and each free organization can have **up to 1,000 members**. You can also downgrade an organization to the Free plan even if it has multiple members. The only limit is the 1,000 free organizations per account; if you reach that, you'll see an error that you've reached the maximum number of free organizations. For details, see [Manage billing](/docs/manage/orgs-manage#manage-billing) and [Neon plans](/docs/introduction/plans).
