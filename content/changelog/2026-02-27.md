## Neon Auth updates and enhancements

You can now configure **Organizations** for Neon Auth from the Neon Console. In your project, open **Auth** in the sidebar and go to the **Configuration** tab to find the new Organizations section. You can enable or disable organizations, set a maximum number of organizations per user, allow or disallow users to create organizations, and choose the default role for organization creators (admin or owner). These settings support multi-tenant apps where users create and join organizations.

## Timescale DB on Postgres 18

We've added support for the [timescaledb](/docs/extensions/timescaledb) extension on Postgres 18. You can now use Timescale DB for time-series data on projects running Postgres 18. Install it with `CREATE EXTENSION timescaledb;` and see [The timescaledb extension](/docs/extensions/timescaledb) for setup and usage.
