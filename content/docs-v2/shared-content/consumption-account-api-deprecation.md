<Admonition type="warning" title="Deprecation notice: GET /consumption_history/account">
The **[Get account consumption metrics](https://api-docs.neon.tech/reference/getconsumptionhistoryperaccount)** endpoint (`GET /consumption_history/account`) is **deprecated**, with a planned sunset of **June 1, 2026**. Migrate to **[Query consumption metrics](/docs/guides/consumption-metrics)** for invoice-aligned project metrics, or to the **[legacy project metrics endpoint](/docs/guides/consumption-metrics-legacy#project-level-endpoint)** (`GET /consumption_history/projects`) if you need legacy metrics per project.
</Admonition>
