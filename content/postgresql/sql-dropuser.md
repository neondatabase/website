## Description

`DROP USER` is simply an alternate spelling of [`DROP ROLE`](sql-droprole).

[#id](#id-1.9.3.143.6)

## Compatibility

The `DROP USER` statement is a PostgreSQL extension. The SQL standard leaves the definition of users to the implementation.
