<Admonition type="warning" title="Deprecation notice: Azure regions on Neon">
**Starting April 2, 2026**: You can no longer create new Neon projects in Azure regions. The `azure-eastus2`, `azure-westus3`, and `azure-gwc` regions are deprecated.

If you are using Azure now, migration deadlines for existing projects will be communicated to you directly by email. Please start planning for a data migration. See [Region migration](/docs/import/region-migration) for supported migration paths.
</Admonition>
