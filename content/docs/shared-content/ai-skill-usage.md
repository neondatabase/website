## How to use

For automatic context in your AI assistant, install the Neon Agent Skills:

```bash
npx add-skill neondatabase/agent-skills
```

This works with:
- **Cursor** - Skills are auto-discovered and invokable via slash commands
- **Claude Code** - Skills load automatically when relevant
- **Other agents** - Any tool supporting the Agent Skills format

Once installed, your AI assistant can access {name} guidance automatically without manual configuration.

### Alternatives

If you prefer to work with individual reference files instead of installing the full skill:

- **Copy from this page** - Use the code reference displayed below
- **Browse the repository** - Download files from the [Agent Skills repository](https://github.com/neondatabase/agent-skills)

See the [Agent Skills documentation](https://agentskills.io) for more installation options and usage patterns.
